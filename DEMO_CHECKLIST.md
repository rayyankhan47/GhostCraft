# Demo Startup Checklist

Run each in a **separate terminal tab**, in this order:

### 1. Minecraft Server
```bash
cd ghostcraft/minecraft-server
java -jar paper.jar nogui
```
Wait for `Done! For help, type "help"`

### 2. GhostCraft Bot
```bash
cd ghostcraft
npm run dev
```
Wait for `RCON connected. All systems ready.`

### 3. Discord
Open Discord → `#ghostcraft-main` → run `/auto I want to build a simple wooden house`
